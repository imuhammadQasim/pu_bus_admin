# 🚌 Punjab University Bus - Admin Portal

A professional, high-performance administrative dashboard designed for the University of the Punjab transport management system. Built with modern web technologies to ensure efficiency, security, and real-time monitoring.

## 🚀 Core Features

- **Dynamic Dashboard**: Real-time visualization of fleet status, student reports, and system performance metrics.
- **Fleet Management**: Comprehensive tracking of all university buses, driver assignments, and maintenance schedules.
- **Route Optimization**: Visual management of bus routes, waypoints, and stops across all PU campuses.
- **Student Report Center**: Centralized moderation system for student feedback, complaints, and service requests.
- **Lost & Found Portal**: Efficient tracking and resolution of items found on the transport network.
- **User Directory**: Secure management of student accounts with university-exclusive email verification (@pu.edu.pk).

## 🛠 Tech Stack

- **Frontend**: React 19 + Vite 8
- **Styling**: Tailwind CSS 4 (Punjab University Theme)
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **State Management**: React Context API
- **Notifications**: Sonner (Rich Toast Notifications)

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Access the Portal**:
   Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛡 Security Note

This portal is protected by administrative guards. To access the dashboard during the demo phase, use university-domain credentials (@pu.edu.pk).
